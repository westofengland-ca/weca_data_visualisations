# weca_data_visualisations
A public repo to contain non-sensitive data visuals for use in GitHub pages

Each visual should be in their own folder as an .html page for GitHub pages to reference. Scripts should sit within the scripts folder.

Data visuals can be made using python `plotly` (`uv pip install plotly`)

## setup

1. install uv
2. `uv venv .venv`
3. `.venv\Scripts\activate`
4. `uv pip install -r requirements.txt`
